# Python Projects

This repository contains a collection of Python projects demonstrating problem-solving, data structures, and core programming concepts developed through coursework and hands-on practice.

## Projects Included

### Anagram Dictionary Generator
Reads words from a file and groups them into anagrams using a dictionary-based approach.  
**Concepts:** File handling, dictionaries, string manipulation, algorithms

### Student Data Management System
Stores and processes student data, including calculating highest, lowest, and average scores.  
**Concepts:** Lists, tuples, data processing, functions

### Population Growth Simulation
Simulates population growth over time and visualizes results using text-based output.  
**Concepts:** Loops, mathematical modeling, data visualization

### Recursive Binary Converter
Converts integers into binary format using recursion.  
**Concepts:** Recursion, problem-solving, algorithm design

### Object-Oriented Management System
Implements classes and inheritance to model management structures.  
**Concepts:** Object-Oriented Programming (OOP), classes, inheritance

## Technologies Used
- Python

## Purpose
These projects were created to develop programming logic, algorithmic thinking, and real-world problem-solving skills. They represent foundational knowledge in software development and computer science concepts.
